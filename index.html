tidydraws - Tidy Bayesian Visualization in Python

A tidybayes-inspired data layer for declarative Bayesian visualization in Python.

Why tidydraws?

  • Extract posterior draws into clean, per-dimension frames
  • Semantically separate parameter space (beta) from prediction space (mu)
  • Lazy evaluation with predicate pushdown for efficient memory usage
  • Declarative syntax matching R's tidybayes approach
  • Supports ArviZ 1.0 DataTrees

Core Functions

  • spread_draws(dt, "var[dim1, dim2]", ...) — Extract posterior draws into a tidy Polars LazyFrame
  • add_epred_draws(dt, newdata=..., var_name="mu") — Join prediction draws to a covariate grid
  • spread_draws_compare(dt, "var[groups]", groups=["posterior", "prior"]) — Stack draws from multiple groups

Installation

Install from PyPI:

pip install tidydraws

Or for development setup:

uv sync

Quick Example

Extract parameter draws:

import arviz as az import tidydraws as td # Load your model dt = az.from_netcdf("model.nc") # Extract draws beta_draws = td.spread_draws(dt, "beta[groups]", "intercept[groups]")

Join predictions to covariates:

pred_draws = td.add_epred_draws(dt, newdata=None, var_name="mu")
